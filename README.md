# ToolkitProo

A professional, fully functional multi-page website for ToolkitProo - a free online tools platform offering financial calculators for mortgage, loans, VAT, ROI, and more.

## Website URL
**Domain:** https://toolkitproo.online

## Features

### Pages
- **Home** - Hero section, features, popular tools, blog preview, CTA
- **Tools** - 9 fully functional calculators organized by category
- **Blog** - 10 SEO-optimized articles on loans, insurance, and home & garden
- **About** - Company information and contact details
- **Contact** - Contact form with email and phone information
- **FAQ** - Frequently asked questions with accordion functionality
- **Legal Pages** - Privacy Policy, Terms & Conditions, Disclaimer

### Functional Calculators
1. **Mortgage Calculator** - Calculate monthly payments, total interest, and amortization
2. **Loan Repayment Calculator** - Plan personal loan repayments
3. **Savings Interest Calculator** - Project savings growth with compound interest
4. **VAT Calculator** - Add or remove VAT from amounts
5. **Invoice Generator** - Create simple professional invoices
6. **Profit Margin Calculator** - Calculate margins and markup percentages
7. **ROI Calculator** - Measure return on investment
8. **Stamp Duty Calculator UK** - Calculate SDLT for property purchases
9. **Rent Affordability Calculator** - Determine affordable rent based on income
10. **Property ROI Calculator** - Analyze property investment returns

## Hosting

This website is designed to be hosted on GitHub Pages or any static hosting service.

### GitHub Pages Setup
1. Create a new repository named `toolkitproo.github.io`
2. Upload all files to the repository
3. Enable GitHub Pages in repository settings
4. Your site will be available at https://toolkitproo.github.io

### Custom Domain
To use toolkitproo.online:
1. Add a CNAME file with your domain name
2. Configure DNS records with your domain registrar:
   - A records pointing to GitHub Pages IPs
   - Or CNAME record pointing to your GitHub Pages URL
